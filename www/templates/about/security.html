Security Policy

It is Fuelergy’s and Frostware’s main goal to make the User feel comfortable within the Application. This includes functionality, usability, accessibility, availability, and very importantly Security. The User should feel comfortable that there are no known vulnerabilities to their sensitive data, and interactions within the application. This policy directs and notifies the User and security measures and prevention from within the Application.

Encrypted Sensitive Data

All data that is deemed sensitive by the Application is secured using the latest cryptography methods. These cryptography measures can be requested by the User at any time. No sensitive data will be stored as entered. Only encrypted sensitive data will be stored to enhance confidentiality and integrity of the Application.

Data Security

Along with out secure data handling, which includes privatizing and encryption data, our databases are secured and provision by Mongolab. Their security policies and administration can be found here: http://docs.mongolab.com/security

Network Security

Our servers are provisioned and maintained by Heroku. You can find server and network security measures devoted by Heroku found here: https://www.heroku.com/policy/security

Secure Physical Location

Our servers are provisioned by Heroku and Mongolab which host within Amazon’s AWS data centers. You can find a portion of their site designed to explain their security measures. http://aws.amazon.com/compliance

Extensive Authentication and Authorization

The Application goes through extensive authentication and authorization tools to make sure that Users are authenticated, but also authorized in the actions they are taking within the Application. Fuelergy at all times will be confident in who the User is, and what they are allowed to do. All Application actions are logged, authenticated, and authorized so that the User is comfortable that they know their data is secure, confidential, availability and held with high integrity.

Experienced Team

Our engineering team is a trusted group of individuals who have dealt with securing and handling extremely sensitive information and serving up usable application with high integrity. Our group continues to employe these standards within the Application and Frostware network. Our operations and assurance teams are dedicated individuals that support our network partnershps, including Heroku and Mongolab, to keep our integration compliant with our security policy and security standards.
